Over the years, The Legend of Zelda: Ocarina of Time has had a storied history with speedrunners. It is amongst the most popular speedrun games in the world due to it's plethora of intricate tricks, glitches and exploits. Through constant pushing of the game engine by an extremely dedicated community, what was once an almost 7 hours speedrun, can now be completed in less than 8 minutes.

This webpage is a visualized history of the world record speedrun times for the video game ‘The Legend of Zelda: Ocarina of Time’. The data is based on a community run spreadsheet which can be found here. While some records may have video links, many were lost in age.

Speedrun Categories

  • Any% - Get to the end by any in-game means.
  • 100% - Complete 100% of the game and reach the end.
  • MST - Complete the Child Dungeons, Temples, and Ganondorf's Castle Trials and get to the end.
  • All Dungeons - Complete all dungeons and get to the end.
  • Glitchless Any% - Get to the end without using any glitches.

Major Glitches/Sequence Breaks

  • RBA - Stands for Reverse Bottle Adventure. It is glitch where by placing a bottle item on B allows us to write the internal item value of a set of bottles outside the space reserved for the inventory.
  • SRM / ACE - Stands for Stale Reference Munipulation / Arbirary Code Execution. It is glitch which allows the player to inject arbitrary code into the game engine.
